## Customer Churn Analysis Report

### Overall Churn Rate
**26.5%** of customers churned.

---

### Key Factors Driving Churn

| Factor | Segment | Churn Rate |
|--------|---------|------------|
| **Contract Type** | Month-to-month | **43%** |
| | One year | 11% |
| | Two year | 3% |
| **Internet Service** | Fiber optic | **30%** |
| | DSL | 19% |
| | No internet | 7% |
| **Payment Method** | Electronic check | **45%** |
| | Other methods | ~15% |
| **Tenure** | 0-12 months | **42%** |
| | 49+ months | 15% |
| **Monthly Charges** | $100+ | **35%** |
| | Under $35 | 10% |
| **Online Security** | No | **41%** |
| | Yes | 14% |
| **Tech Support** | No | **33%** |
| | Yes | 12% |

---

### Summary

1. **Month-to-month contracts** are the strongest predictor of churn
2. **Electronic check payments** correlate with higher churn
3. **Fiber optic customers** churn more than DSL users
4. **First year is critical** - 42% of churn happens in first 12 months
5. **Lack of security/support services** doubles churn rate
6. **Higher monthly charges** ($100+) show elevated churn risk

### Recommendation

Target month-to-month fiber optic customers paying via electronic check with retention offers, especially within first 12 months. Upsell Online Security and Tech Support to reduce churn risk.
